Recursos

Tudo que você precisa para afinar

🎯
Algoritmo YIN
Detecção de pitch com algoritmo YIN em TypeScript puro. Threshold 0.10, janela Hann, interpolação parabólica para máxima precisão.
⚡
Baixa latência
Ring buffer Float32Array com copyWithin. Overlap de 50% e atualização de UI a 30 Hz.
📡
Over-The-Air Update
Atualizações via EAS Update sem publicar nova versão nas lojas. Canais development, preview e production.
🎸
6 cordas padrão
E2 · A2 · D3 · G3 · B3 · E4. Grid interativo com destaque da corda mais próxima da frequência detectada.
📊
Agulha animada
Indicador visual de cents com animação spring nativa. Feedback de cor para grave, afinado e agudo em tempo real.
🔇
Gate de silêncio
RMS gate para ignorar ruído ambiente. EMA simples (α=0.25) suaviza leituras sem mascarar transientes.
